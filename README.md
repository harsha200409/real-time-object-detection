# Real-Time Object Detection Model

## Technologies Used

* Python
* OpenCV
* YOLOv8

## Project Overview

This project performs real-time object detection using a webcam feed. YOLOv8 is used to detect and classify objects, while OpenCV handles video capture and display.

## Features

* Real-time webcam object detection
* Bounding boxes around detected objects
* Confidence score display
* Live video stream processing

## Tools and Libraries

* Python
* OpenCV
* Ultralytics YOLOv8

## Future Improvements

* Improve detection accuracy
* Add object counting
* Save detection results
* Support video file input
